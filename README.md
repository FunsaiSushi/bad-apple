# BadAppleBash

![Design sem nome](https://github.com/user-attachments/assets/329e599e-2284-4d9d-8936-3c166f85334d)

This project reproduces the ["Bad Apple!!"](https://youtu.be/9lNZ_Rnr7Jc?si=ROgXrVvdx13oKPM4) video using Bash script, with all graphics rendered in ASCII art.

## Usage Instructions

### Quick Start (Curl Installation)

Run the animation directly with a single command:

```bash
curl -sSL https://raw.githubusercontent.com/FunsaiSushi/bad-apple/main/install.sh | sh
```

This will automatically download and run the animation. The files are temporarily stored and cleaned up after execution.

### Manual Installation

1.  Clone the repository to your local environment:

```bash
git clone https://github.com/FunsaiSushi/BadAppleBash.git
```

2.  Navigate to the project directory:

```bash
cd BadAppleBash
```

3.  Make the `run.sh` script executable:

```bash
chmod +x run.sh
```

4.  Run the script:

```bash
./run.sh
```
